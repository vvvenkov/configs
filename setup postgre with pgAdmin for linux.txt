You need to set a password for the user:

1. update your system   (curl is needed)
2. postgresql.org
3. pgadmin.org (install either desktop or web version)

4. in pgadmin:
server name and hostname should be localhost
username is postgres

4:
$ sudo -u username psql
(postgres)

inside the shell:
$ alter user postgres password "password";
(or set any new user)

5: Use password in PgAdmin
